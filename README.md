Shiksha-Dost AI Support for Teachers

Project Goal Shiksha-Dost provides instant classroom tips for teachers. It solves Theme 1 by giving immediate help during lessons.

Problem Teachers lack real-time mentorship for classroom challenges. This app provides that support on mobile.

Solution Teachers select a grade and enter a challenge. The app provides a low-cost activity, like the Empty Bowl Game to teach zero.

Tech Stack Python and Streamlit.

Files app.py: Application interface coach.py: Logic and activities requirements.txt: Dependencies

How to Run

Install: pip install streamlit

Run: streamlit run app.py

Roadmap

Current: Stable prototype with fixed activities.

Future: AI integration for custom coaching.

Future: Support for regional languages.
